## 1. 단일 20EMA 백테스팅 전략 (추천)

과거 일별 OHLCV 데이터를 기반으로 실전 전략과 동일한 매수/매도 로직을 시뮬레이션하는 전략입니다.
하이브리드 매도 로직(즉시 손절 + EOD 조건부 trailing stop)을 적용합니다.

### 사용 지표

| 지표 | 용도 |
|-----|------|
| EMA 20 | 단기 추세 판단, 매수/매도 기준선 |
| EMA 120 | 장기 추세 판단, 하락장 필터 |
| ADX / +DI / -DI | 추세 강도 및 방향 판단 |
| ATR (14) | 변동성 기반 손절선, 가격 가드레일 |
| OBV z-score | 거래량 기반 수급 강도 (7일 룩백) |

### 포지션 상태 흐름

```
None (대기)
    │
    ├─ 1차 매수 조건 충족 ──→ BUY_COMPLETE (1차 매수)
    │                            │
    │                            ├─ 2차 매수 조건 충족 ──→ BUY_COMPLETE (2차 매수)
    │                            │
    │                            ├─ [1차 방어선] 즉시 손절 ──→ None
    │                            │
    │                            └─ [2차 방어선] EOD 1차 매도 ──→ SELL_PRIMARY (잔량 보유)
    │                                                                │
    │                                                                ├─ 재진입 매수 ──→ BUY_COMPLETE
    │                                                                ├─ EOD 2차 전량 매도 ──→ None
    │                                                                └─ [1차 방어선] 즉시 손절 ──→ None
```

---

### 매수 전략

#### 공통 필터
- **하락장 필터**: 20 EMA가 120 EMA 아래에 있으면 신규 진입 및 2차 매수 모두 금지
- **전일 양봉**: 전일 종가 > 전전일 종가 (1차, 2차 매수 공통)

#### 1차 매수

공통 필터 통과 후, 아래 **5가지 조건 모두** 충족 시 진입 (종가 기준)

1. **EMA 근접**: 종가 ≥ EMA20 × 0.995 (EMA 0.5% 이내)
2. **수급 강도**: OBV z-score > 0 AND 2일전 대비 상승 (수급 개선 추세)
3. **괴리율 필터**: 괴리율 ≤ 5% (고점 매수 방지)
4. **추세 방향**: +DI > -DI AND ADX > 20 (상승 추세 + 추세 강도 확인)
5. **전일 양봉**: 전일 종가 > 전전일 종가

#### 2차 매수

1차 매수 후, 공통 필터 통과 후 아래 **시나리오 A 또는 B** 중 하나 충족 시 추가 매수

**시나리오 A - 추세 강화형**
- 가격 가드레일: EMA + ATR×0.3 ≤ 종가 ≤ EMA + ATR×2.0
- ADX > 25 (강한 추세 지속)
- +DI > -DI (상승 방향 유지)
- OBV z-score ≥ 1.2 (수급 지속)

**시나리오 B - 눌림목 반등**
- 가격 가드레일: EMA - ATR×0.5 ≤ 종가 ≤ EMA + ATR×0.3
- 18 ≤ ADX ≤ 23 (조정 구간, 추세 유지)
- +DI > -DI (상승 방향 유지)
- OBV z-score > 0.5 (중립 이상)
- 당일 저가 대비 0.4% 이상 회복

### 매도 전략

포지션 보유 시 매일 **고점(peak_price)** 을 갱신하며, 1차 → 2차 방어선 순서로 매도 조건을 체크합니다.

#### [1차 방어선] 즉시 손절 (일일 저가 기준) — 급락 대응

- **EMA-ATR 동적 손절**: 저가 ≤ EMA20 - ATR×1.0 도달 시 전량 즉시 매도
- 포지션 상태(BUY_COMPLETE / SELL_PRIMARY) 무관하게 항상 최우선 적용
- 발동 시 peak_price 초기화

#### [2차 방어선] EOD 조건부 trailing stop (일일 종가 기준) — 점진적 하락 대응

**게이트 조건** (아래 2가지 **모두** 충족 시에만 고점 대비 하락률 평가):

| 조건 | 판단 기준 | 의미 |
|-----|----------|------|
| 추세 약화 | (+DI - -DI) 격차 2일 연속 감소 | 매수세 대비 매도세의 힘의 균형이 이동 |
| 수급 약화 | OBV z-score가 2일전보다 감소 | 거래량 기반 수급 악화 |

> **DI 격차를 사용하는 이유:**
> - +DI, -DI는 이미 **14일 Wilder 스무딩**이 적용된 값이므로, 2일 연속 격차 감소는 단기 노이즈가 아닌 실질적 방향성 변화를 의미
> - `-DI > +DI` 역전은 강한 상승 이후 지연이 심함 (ex: +10% 급등 후 14일 스무딩으로 인해 역전까지 10일 이상 소요)
> - 격차 감소는 역전 이전에 감지 가능하여 반응 속도 개선

**매도 결정** (게이트 조건 충족 후, 고점 대비 하락률로 단계 결정):

| 현재 상태 | 하락률 기준 | 매도 행동 |
|----------|-----------|----------|
| BUY_COMPLETE | 고점 대비 ≥ **5%** 하락 | 1차 분할 매도 (`sell_ratio%`), SELL_PRIMARY로 전환 |
| SELL_PRIMARY | 고점 대비 ≥ **8%** 하락 | 2차 전량 매도, None(대기)으로 복귀, peak_price 초기화 |

**오매도 방지 (3중 필터):**

| 시나리오 | DI 격차 감소 | OBV 감소 | drawdown ≥5% | 결과 |
|---------|:---:|:---:|:---:|------|
| 진짜 하락 | O | O | O | 매도 발동 |
| 매집 눌림 (세력 매수) | O | **X** (매수세) | O | 매도 안 됨 |
| 일시 조정 (1~2일 하락) | O | O | **X** | 매도 안 됨 |
| 횡보 구간 | **X** | X | X | 매도 안 됨 |

### 알려진 제한사항

1. **일봉 데이터 한계**
   - 장중 EMA-ATR 손절 도달 시점을 정확히 파악 불가
   - 실제 손절가가 아닌 EMA-ATR 계산값으로 대체 처리

2. **ADX/DI 초기 데이터**
   - ADX는 14일 계산 기간 필요, 데이터 초기 구간에서는 필터 미작동
   - 2차 방어선은 2일 연속 DI 비교가 필요하므로 최소 3일차부터 동작

3. **trailing stop 특성**
   - 고점 갱신이 없는 횡보장에서는 drawdown이 누적되어 조기 매도 가능
   - DI 격차 감소 + OBV 감소 게이트가 횡보/매집 구간 오발동 억제

## 2. 이평선 골든크로스 전략

단기/중기/장기 이평선 교차 신호 기반의 단순 전략입니다.

### 전략 개요

- **기술 지표**: 단기/중기/장기 EMA
- **진입**: 단기 EMA > 중기 EMA > 장기 EMA (골든크로스)
- **청산**: 단기 EMA < 중기 EMA (데드크로스)
- **거래 빈도**: 연 5~10회 예상

### 제한사항

- 골든크로스 신호 지연 (후행성)
- 횡보장에서 잦은 거짓 신호 발생 (whipsaw)
- 수급 지표 미반영
- 변동성 기반 손절 없음

---

## 3. 일목균형표 전략

일목균형표 복합 신호 기반 전략입니다.

### 전략 개요

- **기술 지표**: 전환선, 기준선, 선행스팬, 후행스팬
- **진입**: 전환선 > 기준선 (단기 강세) + 현재가 > 선행스팬 (구름대 돌파)
- **청산**: 전환선 < 기준선 (단기 약세) + 현재가 < 선행스팬 (구름대 이탈)
- **거래 빈도**: 연 8~12회 예상

### 제한사항

- 선행스팬 26일 후행으로 신호 지연
- 급변하는 시장에서 신호 늦음
- 수급 지표 미반영
- 변동성 기반 손절 없음

## 실전 전략 vs 백테스팅 전략 비교

| 구분 | 실전 전략 (SingleEMAStrategy) | 백테스팅 전략 (SingleEMABacktestStrategy) |
|------|-------------------------------|------------------------------------------|
| **데이터** | 실시간 5분 간격 체크 | 일별 OHLCV 데이터 |
| **1차 매수** | 동일 (하락장 필터, 5개 조건, 연속 2회) | 동일 (연속 1회) |
| **2차 매수** | 동일 (시나리오 A/B, 1차 매수 후 20분 경과) | 동일 (시나리오 A/B) |
| **즉시 손절** | 실시간 현재가 기준 | 일일 저가 기준 |
| **EOD 매도** | ✅ 사용 (장 마감 후 DB 신호 기반, 3신호 누적) | ✅ 사용 (조건부 trailing stop, DI격차+OBV+하락률) |
| **분할 매도** | ✅ 있음 (SELL_PARTIAL → SELL_ALL) | ✅ 있음 (SELL_PRIMARY → SELL_ALL) |
| **포지션 상태** | SIGNAL 0→1→2→3→0 | None → BUY_COMPLETE → SELL_PRIMARY → None |
| **주요 목적** | 장중 실시간 매매 | 과거 데이터 성능 검증 |
