## 1. 단일 20EMA 백테스팅 전략 (추천)

과거 일별 OHLCV 데이터를 기반으로 실전 전략과 동일한 매수/매도 로직을 시뮬레이션하는 전략입니다.
실전 전략의 하이브리드 매도 로직(즉시 손절 + EOD 분할 매도)을 그대로 반영합니다.

### 사용 지표

| 지표 | 용도 |
|-----|------|
| EMA 20 | 단기 추세 판단, 매수/매도 기준선 |
| EMA 120 | 장기 추세 판단, 하락장 필터 |
| ADX / +DI / -DI | 추세 강도 및 방향 판단 |
| ATR (14) | 변동성 기반 손절선, 가격 가드레일 |
| OBV z-score | 거래량 기반 수급 강도 (7일 룩백) |

### 포지션 상태 흐름

```
None (대기)
    │
    ├─ 1차 매수 조건 충족 ──→ BUY_COMPLETE (1차 매수)
    │                            │
    │                            ├─ 2차 매수 조건 충족 ──→ BUY_COMPLETE (2차 매수)
    │                            │
    │                            ├─ [1차 방어선] 즉시 손절 ──→ None
    │                            │
    │                            └─ [2차 방어선] EOD 1차 매도 ──→ SELL_PRIMARY (잔량 보유)
    │                                                                │
    │                                                                ├─ 재진입 매수 ──→ BUY_COMPLETE
    │                                                                ├─ EOD 2차 전량 매도 ──→ None
    │                                                                └─ [1차 방어선] 즉시 손절 ──→ None
```

---

### 매수 전략

#### 하락장 필터
- 20 EMA가 120 EMA 아래에 있으면 신규 진입 및 2차 매수 모두 금지

#### 1차 매수

아래 **4가지 조건 모두** 충족 시 진입 (종가 기준)

1. **EMA 근접**: 종가 ≥ EMA20 × 0.995 (EMA 0.5% 이내)
2. **수급 강도**: OBV z-score > 1.0
3. **괴리율 필터**: 괴리율 ≤ 5% (고점 매수 방지)
4. **추세 방향**: +DI > -DI (상승 추세 확인)

#### 2차 매수

1차 매수 후, 아래 **시나리오 A 또는 B** 중 하나 충족 시 추가 매수

**시나리오 A - 추세 강화형**
- 가격 가드레일: EMA + ATR×0.3 ≤ 종가 ≤ EMA + ATR×2.0
- ADX > 25 (강한 추세 지속)
- +DI > -DI (상승 방향 유지)
- OBV z-score ≥ 1.2 (수급 지속)

**시나리오 B - 눌림목 반등**
- 가격 가드레일: EMA - ATR×0.5 ≤ 종가 ≤ EMA + ATR×0.3
- 18 ≤ ADX ≤ 23 (조정 구간, 추세 유지)
- +DI > -DI (상승 방향 유지)
- OBV z-score > 0 (중립 이상)
- 당일 저가 대비 0.4% 이상 회복

---

### 매도 전략

#### [1차 방어선] 즉시 손절 (일일 저가 기준)

- **EMA-ATR 동적 손절**: 저가 ≤ EMA20 - ATR×1.0 도달 시 전량 즉시 매도
- 포지션 상태(BUY_COMPLETE / SELL_PRIMARY) 무관하게 항상 최우선 적용

#### [2차 방어선] EOD 매도 (일일 종가 기준)

종가 기준으로 아래 3가지 EOD 신호를 누적 추적합니다.
**시간 윈도우**: 최근 3거래일 이내 발생한 신호만 유효 (3일 경과 시 자동 소멸)

**3가지 EOD 신호:**

| 신호 | 조건 |
|-----|------|
| EMA 이탈 | 종가 < EMA20 |
| 추세 약화 | ADX < 20 AND -DI > +DI (2일 연속) |
| 수급 이탈 | OBV z-score < -1.0 |

**1차 분할 매도 (SELL_PRIMARY)**
- BUY_COMPLETE 상태에서 3개 신호 중 **2개 이상** 충족 시
- `sell_ratio%` 분할 매도, 잔량 보유 후 SELL_PRIMARY 상태로 전환
- 신호 기록 초기화

**2차 전량 매도**
- SELL_PRIMARY 상태에서 3개 신호 **모두** 충족 시
- 잔량 전량 매도 후 None(대기) 상태로 복귀

---

### 거래 비용

| 항목 | 비율 |
|-----|------|
| 매수 수수료 | 0.147% |
| 매도 수수료 | 0.147% |
| 증권거래세 | 0.20% |

### 주요 파라미터

| 파라미터 | 값 | 설명 |
|---------|-----|------|
| EMA_PERIOD | 20 | 단기 EMA 기간 |
| EMA_LONG_PERIOD | 120 | 장기 EMA 기간 (하락장 판단) |
| OBV_Z_BUY_THRESHOLD | 1.0 | 1차 매수 OBV 기준 |
| OBV_LOOKBACK | 7 | OBV z-score 계산 기간 |
| MAX_GAP_RATIO | 0.05 | 최대 괴리율 (5%) |
| ATR_MULTIPLIER | 1.0 | 즉시 손절 ATR 배수 |
| EOD_SIGNAL_WINDOW_DAYS | 3 | EOD 신호 유효 기간 (3일) |
| TREND_BUY_ATR_LOWER | 0.3 | 시나리오 A 하한 배수 |
| TREND_BUY_ATR_UPPER | 2.0 | 시나리오 A 상한 배수 |
| TREND_BUY_OBV_THRESHOLD | 1.2 | 시나리오 A OBV 기준 |
| TREND_BUY_ADX_MIN | 25 | 시나리오 A ADX 최소값 |
| PULLBACK_BUY_ATR_LOWER | -0.5 | 시나리오 B 하한 배수 |
| PULLBACK_BUY_ATR_UPPER | 0.3 | 시나리오 B 상한 배수 |
| PULLBACK_BUY_ADX_MIN | 18 | 시나리오 B ADX 하한 |
| PULLBACK_BUY_ADX_MAX | 23 | 시나리오 B ADX 상한 |
| PULLBACK_BUY_OBV_MIN | 0.5 | 시나리오 B OBV 기준 (실전과 동일) |
| PULLBACK_BUY_REBOUND_RATIO | 1.004 | 시나리오 B 저가 반등 비율 (0.4%) |

---

### 알려진 제한사항

1. **일봉 데이터 한계**
   - 장중 EMA-ATR 손절 도달 시점을 정확히 파악 불가
   - 실제 손절가가 아닌 EMA-ATR 계산값으로 대체 처리

2. **EOD 신호 지연**
   - 1차 방어선 미발동 시 분할 매도 구조로 인해 잔량이 며칠간 보유될 수 있음
   - 시간 윈도우(3일) 내에 2차 전량 매도 신호가 미발생하면 잔량 장기 보유 가능

3. **ADX 초기 데이터**
   - ADX는 14일 계산 기간 필요, 데이터 초기 구간에서는 필터 미작동

## 2. 이평선 골든크로스 전략

단기/중기/장기 이평선 교차 신호 기반의 단순 전략입니다.

### 전략 개요

- **기술 지표**: 단기/중기/장기 EMA
- **진입**: 단기 EMA > 중기 EMA > 장기 EMA (골든크로스)
- **청산**: 단기 EMA < 중기 EMA (데드크로스)
- **거래 빈도**: 연 5~10회 예상

### 제한사항

- 골든크로스 신호 지연 (후행성)
- 횡보장에서 잦은 거짓 신호 발생 (whipsaw)
- 수급 지표 미반영
- 변동성 기반 손절 없음

## 3. 일목균형표 전략

일목균형표 복합 신호 기반 전략입니다.

### 전략 개요

- **기술 지표**: 전환선, 기준선, 선행스팬, 후행스팬
- **진입**: 전환선 > 기준선 (단기 강세) + 현재가 > 선행스팬 (구름대 돌파)
- **청산**: 전환선 < 기준선 (단기 약세) + 현재가 < 선행스팬 (구름대 이탈)
- **거래 빈도**: 연 8~12회 예상

### 제한사항

- 선행스팬 26일 후행으로 신호 지연
- 급변하는 시장에서 신호 늦음
- 수급 지표 미반영
- 변동성 기반 손절 없음

## 실전 전략 vs 백테스팅 전략 비교

| 구분 | 실전 전략 (SingleEMAStrategy) | 백테스팅 전략 (SingleEMABacktestStrategy) |
|------|-------------------------------|------------------------------------------|
| **데이터** | 실시간 5분 간격 체크 | 일별 OHLCV 데이터 |
| **1차 매수** | 동일 (하락장 필터, 4개 조건, 연속 2회) | 동일 (연속 1회) |
| **2차 매수** | 동일 (시나리오 A/B, 1차 매수 후 20분 경과) | 동일 (시나리오 A/B) |
| **즉시 손절** | 실시간 현재가 기준 | 일일 저가 기준 |
| **EOD 매도** | ✅ 사용 (장 마감 후 DB 신호 기반) | ✅ 사용 (일별 종가 계산, 시간 윈도우 3일) |
| **분할 매도** | ✅ 있음 (SELL_PARTIAL → SELL_ALL) | ✅ 있음 (SELL_PRIMARY → SELL_ALL) |
| **포지션 상태** | SIGNAL 0→1→2→3→0 | None → BUY_COMPLETE → SELL_PRIMARY → None |
| **주요 목적** | 장중 실시간 매매 | 과거 데이터 성능 검증 |
