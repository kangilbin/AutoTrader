## 1. 단일 20EMA 백테스팅 전략 (추천)

과거 일별 OHLCV 데이터를 기반으로 실전 전략과 동일한 매수/매도 로직을 시뮬레이션하는 전략입니다.
하이브리드 매도 로직(즉시 손절 + 조건부 trailing stop)을 적용합니다.

### 사용 지표

| 지표 | 용도 |
|-----|------|
| EMA 20 | 단기 추세 판단, 매수/매도 기준선 |
| ADX / +DI / -DI | 추세 강도 및 방향 판단 |
| ATR (14) | 변동성 기반 손절선, 가격 가드레일 |
| OBV z-score | 거래량 기반 수급 강도 (7일 룩백) |

### 포지션 상태 흐름

```
None (대기)
    │
    ├─ 1차 매수 조건 충족 ──→ BUY_COMPLETE (1차 매수)
    │                            │
    │                            ├─ 2차 매수 조건 충족 ──→ BUY_COMPLETE (2차 매수)
    │                            │
    │                            ├─ [1차 방어선] 즉시 손절 ──→ None
    │                            │
    │                            └─ [2차 방어선] 익절 ──→ None
```

---

### 매수 전략

#### 공통 필터
- **전일 양봉**: 전일 종가 >= 전일 시가 (양봉 캔들, 1차/2차 매수 공통)
- **급등 필터**: 당일 변동률(절대값) ≤ 5% (급등/급락 종목 배제)

#### 1차 매수

공통 필터 통과 후, **시나리오 A 또는 B** 중 하나 충족 시 진입 (종가 기준)

**시나리오 A — 눌림목 매집 진입**
- **가격 범위**: EMA20 ± ATR×0.5 이내
- **수급 개선**: OBV z-score > 0 AND 전일 대비 상승
- **추세 존재**: ADX 18~30 + +DI > -DI + EMA 상승 중 (전일 대비)

**시나리오 B — 추세 추종 돌파 진입**
- **EMA 상승 추세**: 금일 EMA20 > 전일 EMA20
- **가격 확인**: 종가 > EMA20 (괴리율 3% 이내)
- **추세 방향**: +DI > -DI + ADX > 15
- **수급 동반**: OBV z-score > 0

#### 2차 매수

1차 매수 후, 아래 조건 **모두** 충족 시 추가 매수 (종가 기준)

- **전일 양봉**: 전일 종가 >= 전일 시가
- **가격 위치**: EMA + ATR×0.5 ≤ 종가 ≤ EMA + ATR×2.0 (추세 확인 + 과열 방지)
- **추세 안정**: ADX >= 20 + +DI > -DI
- **수급 확인**: OBV z-score >= 0.5

---

### 매도 전략

포지션 보유 시 매일 **고가(STCK_HGPR)** 기준으로 **고점(peak_price)** 을 갱신하며, 1차 → 2차 방어선 순서로 매도 조건을 체크합니다.

#### [1차 방어선] 즉시 손절 (일일 저가 기준) — 급락 대응

- **EMA-ATR 동적 손절**: 저가 ≤ EMA20 - ATR×1.0 도달 시 전량 즉시 매도
- 포지션 상태(BUY_COMPLETE) 무관하게 항상 최우선 적용
- ATR=0인 경우 손절 체크 스킵
- 발동 시 peak_price 초기화

#### [2차 방어선] trailing stop 익절 (일일 저가 기준)

- **고점 대비 ATR×2.0 하락 시 전량 매도**: 저가 ≤ 고점(peak_price) - ATR×2.0
- 게이트 조건 없음 (ATR 자체가 변동성 적응형 필터 역할)
- ATR 무효 시 폴백: 고점 대비 5.0% 하락

| 파라미터 | 값 | 설명 |
|---------|-----|------|
| **공통 지표** | | |

| OBV_LOOKBACK | 7 | OBV z-score 계산 기간 |
| **1차 매수** | | |

| SECOND_BUY_ATR_UPPER | 2.0 | 상한 배수 (EMA + ATR×2.0) |
| SECOND_BUY_ADX_MIN | 20 | ADX 최소값 |

---

### 알려진 제한사항

1. **일봉 데이터 한계**
   - 장중 EMA-ATR 손절 도달 시점을 정확히 파악 불가
   - 실제 손절가가 아닌 EMA-ATR 계산값으로 대체 처리

2. **ADX/DI 초기 데이터**
   - ADX는 14일 계산 기간 필요, 데이터 초기 구간에서는 필터 미작동
   - 2차 방어선은 2일 연속 DI 비교가 필요하므로 최소 3일차부터 동작

3. **trailing stop 특성**
   - 고점 갱신이 없는 횡보장에서는 drawdown이 누적되어 조기 매도 가능
   - DI 격차 감소 + OBV 감소 게이트가 횡보/매집 구간 오발동 억제

## 2. 이평선 골든크로스 전략

단기/중기/장기 이평선 교차 신호 기반의 단순 전략입니다.

### 전략 개요

- **기술 지표**: 단기/중기/장기 EMA
- **진입**: 단기 EMA > 중기 EMA > 장기 EMA (골든크로스)
- **청산**: 단기 EMA < 중기 EMA (데드크로스)
- **거래 빈도**: 연 5~10회 예상

### 제한사항

- 골든크로스 신호 지연 (후행성)
- 횡보장에서 잦은 거짓 신호 발생 (whipsaw)
- 수급 지표 미반영
- 변동성 기반 손절 없음

---

## 3. 일목균형표 전략

일목균형표 복합 신호 기반 전략입니다.

### 전략 개요

- **기술 지표**: 전환선, 기준선, 선행스팬, 후행스팬
- **진입**: 전환선 > 기준선 (단기 강세) + 현재가 > 선행스팬 (구름대 돌파)
- **청산**: 전환선 < 기준선 (단기 약세) + 현재가 < 선행스팬 (구름대 이탈)
- **거래 빈도**: 연 8~12회 예상

### 제한사항

- 선행스팬 26일 후행으로 신호 지연
- 급변하는 시장에서 신호 늦음
- 수급 지표 미반영
- 변동성 기반 손절 없음

## 실전 전략 vs 백테스팅 전략 비교

| 구분 | 실전 전략 (SingleEMAStrategy) | 백테스팅 전략 (SingleEMABacktestStrategy) |
|------|-------------------------------|------------------------------------------|
| **데이터** | 실시간 5분 간격 체크 | 일별 OHLCV 데이터 |
| **1차 매수** | 시나리오 A(눌림목) + B(추세 추종), 연속 2회 | 동일 (연속 1회) |
| **2차 매수** | 통합 조건 (20분 경과 + 전일 양봉) | 동일 (전일 양봉) |
| **즉시 손절** | 실시간 현재가 기준 | 일일 저가 기준 |
| **2차 방어선 (익절)** | 장중 trailing stop (5분마다 체크) | EOD trailing stop (저가 기준) |
| **익절 조건** | 현재가 ≤ 고점 - ATR×2.0 | 저가 ≤ 고점 - ATR×2.0 |
| **PEAK_PRICE 갱신** | 장중 5분마다 고가(stck_hgpr) 기준 | 일일 고가(STCK_HGPR) 기준 |
| **포지션 상태** | SIGNAL 0→1→2→0 | None → BUY_COMPLETE → None |
| **주요 목적** | 장중 실시간 매매 | 과거 데이터 성능 검증 |
