## 1. 단일 20EMA 백테스팅 전략 (추천)

과거 일별 OHLCV 데이터를 기반으로 실전 전략과 동일한 매수/매도 로직을 시뮬레이션하는 전략입니다.
하이브리드 매도 로직(즉시 손절 + 조건부 trailing stop)을 적용합니다.

### 사용 지표

| 지표 | 용도 |
|-----|------|
| EMA 20 | 단기 추세 판단, 매수/매도 기준선 |
| ADX / +DI / -DI | 추세 강도 및 방향 판단 |
| ATR (14) | 변동성 기반 손절선, 가격 가드레일 |
| OBV z-score | 거래량 기반 수급 강도 (7일 룩백) |

### 포지션 상태 흐름

```
signal = 0 (대기)
    │ 매수 (리스크 기반 수량 계산)
    ▼
signal = 1 (보유 - 1차 익절 전) ← peak_price = 매수가
    │ 매일: peak_price = max(peak_price, 고가)
    ├─ [손절] EMA-ATR ──────────→ signal = 0 (전량 매도)
    │
    └─ [1차 익절] PEAK-ATR×2.0 → 50% 매도 → signal = 2
                                              │ peak_price 리셋
                                              ├─ [손절] Max(EMA-ATR, 평단가) → signal = 0 (본전 방어)
                                              └─ [2차 익절] PEAK-ATR×2.0 AND OBV-Z<-0.5 → signal = 0
```

---

### 포지션 사이징 (고정 비율 + 손실 안전장치)

```
Qty = min(
    Capital × ENTRY_PCT / 신호가,              ← 기본: 배정금의 50% 투입
    Capital × MAX_LOSS_PCT / (신호가 - 손절가)  ← 안전장치: 손절 시 배정금의 10% 이내 손실
)
신호가 = EMA20 (저가 ≤ EMA20 ≤ 고가일 때), 범위 밖이면 종가 fallback
손절가 = 진입일 EMA20 - ATR × 1.0
```

---

### 매수 전략

#### 공통 필터
- **급등 필터**: 당일 변동률(절대값) ≤ 5% (급등/급락 종목 배제)

공통 필터 통과 후, **시나리오 A 또는 B** 중 하나 충족 시 진입 (신호가 기준: EMA20, 범위 밖이면 종가 fallback)

**시나리오 A — 눌림목 매집 진입**
- **가격 범위**: EMA20 ± ATR×0.5 이내
- **수급 개선**: OBV z-score > 0 AND 전일 대비 상승
- **추세 존재**: ADX 18~30 + +DI > -DI

**시나리오 B — 추세 추종 돌파 진입**
- **가격 확인**: 종가 > EMA20 (괴리율 3% 이내)
- **추세 방향**: +DI > -DI + ADX > 15
- **수급 동반**: OBV z-score > 0

---

### 매도 전략

#### [손절] 즉시 매도 (일일 저가 기준)

- **signal 1**: 저가 ≤ EMA20 - ATR×1.0 시 전량 매도
- **signal 2**: 저가 ≤ Max(EMA20 - ATR×1.0, 평단가) 시 전량 매도 (**본전 방어**)

#### [1차 익절] 50% 매도 (signal 1 → 2)

- 저가 ≤ 고점(peak_price) - ATR×2.0
- 보유 물량의 **50%** 를 stop_price(고점 - ATR×2.0)에 매도하여 수익 확정
- 매도 후 peak_price를 stop_price로 리셋

#### [2차 익절] 잔량 전량 매도 (signal 2 → 0)

- 저가 ≤ 고점(peak_price) - ATR×2.0 **AND** OBV z-score < -0.5
- stop_price(고점 - ATR×2.0)에 잔량 전량 매도
- OBV 양호 시 매도 안 함 → 추세 추적

| 항목 | 비율 |
|-----|------|
| 매수 수수료 | 0.147% |
| 매도 수수료 | 0.147% |
| 증권거래세 | 0.20% |

---

### 주요 파라미터

| 파라미터 | 값 | 설명 |
|---------|-----|------|
| **공통 지표** | | |
| EMA_PERIOD | 20 | 단기 EMA 기간 |
| OBV_LOOKBACK | 7 | OBV z-score 계산 기간 |
| **매수** | | |
| MAX_SURGE_RATIO | 0.05 | 전일 대비 최대 급등률 (5%) |
| **포지션 사이징** | | |
| ENTRY_PCT | 0.5 | 매 사이클 배정금 대비 투입 비율 (50%) |
| MAX_LOSS_PCT | 0.10 | 손절 시 배정금 대비 최대 손실률 (10%, 고변동성 안전장치) |
| **손절** | | |
| ATR_MULTIPLIER | 1.0 | 손절 ATR 배수 (EMA - ATR×1.0) |
| **익절** | | |
| TRAILING_STOP_ATR_MULT | 2.0 | 고점 대비 ATR 배수 (PEAK - ATR×2.0) |
| FIRST_PROFIT_TAKE_RATIO | 0.5 | 1차 익절 매도 비율 (50%) |
| OBV_Z_SELL_THRESHOLD | -0.5 | 2차 익절 OBV z-score 게이트 |
| TRAILING_STOP_FALLBACK_PCT | 5.0 | ATR 무효 시 폴백 (%) |

---

### 알려진 제한사항

1. **일봉 데이터 한계**
   - 장중 EMA-ATR 손절 도달 시점을 정확히 파악 불가
   - 실제 손절가가 아닌 EMA-ATR 계산값으로 대체 처리

2. **ADX/DI 초기 데이터**
   - ADX는 14일 계산 기간 필요, 데이터 초기 구간에서는 필터 미작동

3. **trailing stop 특성**
   - 고점 갱신이 없는 횡보장에서는 drawdown이 누적되어 조기 매도 가능

## 2. 이평선 골든크로스 전략

단기/중기/장기 이평선 교차 신호 기반의 단순 전략입니다.

### 전략 개요

- **기술 지표**: 단기/중기/장기 EMA
- **진입**: 단기 EMA > 중기 EMA > 장기 EMA (골든크로스)
- **청산**: 단기 EMA < 중기 EMA (데드크로스)
- **거래 빈도**: 연 5~10회 예상

### 제한사항

- 골든크로스 신호 지연 (후행성)
- 횡보장에서 잦은 거짓 신호 발생 (whipsaw)
- 수급 지표 미반영
- 변동성 기반 손절 없음

---

## 3. 일목균형표 전략

일목균형표 복합 신호 기반 전략입니다.

### 전략 개요

- **기술 지표**: 전환선, 기준선, 선행스팬, 후행스팬
- **진입**: 전환선 > 기준선 (단기 강세) + 현재가 > 선행스팬 (구름대 돌파)
- **청산**: 전환선 < 기준선 (단기 약세) + 현재가 < 선행스팬 (구름대 이탈)
- **거래 빈도**: 연 8~12회 예상

### 제한사항

- 선행스팬 26일 후행으로 신호 지연
- 급변하는 시장에서 신호 늦음
- 수급 지표 미반영
- 변동성 기반 손절 없음

## 실전 전략 vs 백테스팅 전략 비교

| 구분 | 실전 전략 (SingleEMAStrategy) | 백테스팅 전략 (SingleEMABacktestStrategy) |
|------|-------------------------------|------------------------------------------|
| **데이터** | 실시간 5분 간격 체크 | 일별 OHLCV 데이터 |
| **1차 매수** | 시나리오 A(눌림목) + B(추세 추종), 연속 2회 | 동일 (연속 1회), 신호가(EMA20) 기준 체결 |
| **2차 매수** | 통합 조건 (20분 경과 + 전일 양봉) | 동일 (전일 양봉) |
| **포지션 사이징** | min(CUR_AMOUNT×ENTRY_PCT/Price, CUR_AMOUNT×MAX_LOSS_PCT/Risk) | 동일 |
| **손절** | 실시간 현재가 기준 | 저가 ≤ 손절가 시 손절가로 체결 |
| **1차 익절** | 현재가 ≤ PEAK - ATR×2.0 → 50% 매도 | 저가 ≤ stop_price 시 stop_price로 체결 |
| **2차 익절** | ATR×2.0 + OBV-Z < -0.5 → 잔량 매도 | stop_price로 체결 (OBV 게이트 동일) |
| **본전 방어** | SIGNAL 2: Max(EMA-ATR, 평단가) | 동일 |
| **포지션 상태** | SIGNAL 0→1→2→0 | signal 0→1→2→0 |
| **주요 목적** | 장중 실시간 매매 | 과거 데이터 성능 검증 |
